（电脑走黑棋）
等到电脑走时，电脑尝试这一步的所有可能的走法。每一种走法，电脑记录
自己黑棋的分数 blackTotalScore=所有活着的黑棋分数的和，
红棋的分数 redTotalScore=所有活着的红棋分数的和
那么局面分 score = blackTotalScore - redTotalScore
然后选取所有走法中，局面分最大的那种走法。

```cpp
if(score>maxScore)
{
    maxScore=score;
    ret=step;
}
return ret;
```
